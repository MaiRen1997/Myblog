## 第一步：安装 uv

### macOS / Linux

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### 验证安装

```bash
uv --version
```

## 第二步：创建项目目录并初始化

```bash
# 1. 创建项目目录
mkdir my-awesome-project
cd my-awesome-project

# 2. 初始化项目
uv init
```

执行后，`uv` 会自动生成以下文件：

```text
my-awesome-project/
├── .gitignore          # Git 忽略文件
├── .python-version     # Python 版本锁定
├── README.md           # 项目说明
├── main.py             # 示例主程序
├── pyproject.toml      # 项目配置文件
└── uv.lock             # 依赖锁定文件
```

查看生成的 `pyproject.toml`：

```
[project]
name = "my-awesome-project"
version = "0.1.0"
description = "Add your description here"
readme = "README.md"
requires-python = ">=3.12"
dependencies = []
```

## 第三步：创建虚拟环境

`uv init` 只是生成了配置文件，还没有创建虚拟环境。创建虚拟环境：

```bash
# 创建虚拟环境（默认在 .venv 目录）
uv venv

# 或者指定 Python 版本
uv venv --python 3.12
```

输出示例：

```text
Using Python 3.12.0
Creating virtual environment at: .venv
Activate with: source .venv/bin/activate
```

现在项目结构变成了：

```text
my-awesome-project/
├── .venv/              # 虚拟环境目录（新增）
├── .gitignore
├── .python-version
├── README.md
├── main.py
├── pyproject.toml
└── uv.lock
```

------

## 第四步：同步依赖

虽然还没有添加依赖，但执行 `uv sync` 可以确保环境同步：

```bash
uv sync
```

这会：

1. 检查虚拟环境是否存在
2. 根据 `pyproject.toml` 安装依赖（目前为空）
3. 更新 `uv.lock` 文件

------

## 第五步：添加项目依赖

假设我们要创建一个 Web API 项目，添加 FastAPI 和 Uvicorn：

```bash
# 添加运行时依赖
uv add fastapi uvicorn

# 添加开发依赖（用于测试和代码格式化）
uv add pytest black --dev
```

查看更新后的 `pyproject.toml`：

```text
[project]
name = "my-awesome-project"
version = "0.1.0"
description = "Add your description here"
readme = "README.md"
requires-python = ">=3.12"
dependencies = [
    "fastapi>=0.115.0",
    "uvicorn[standard]>=0.34.0",
]

[project.optional-dependencies]
dev = [
    "pytest>=8.3.0",
    "black>=24.10.0",
]
```

## 第六步：编写代码

修改 `main.py`，添加一个简单的 FastAPI 应用：

python

```
# main.py
from fastapi import FastAPI

app = FastAPI(title="My Awesome API", version="0.1.0")

@app.get("/")
def read_root():
    return {"message": "Hello, World!"}

@app.get("/health")
def health_check():
    return {"status": "healthy"}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

------

## 第七步：运行项目

使用 `uv run` 在虚拟环境中运行代码：

### 开发模式运行

```bash
# 直接运行
uv run python main.py

# 或使用 uvicorn 命令
uv run uvicorn main:app --reload
```

### 访问应用

打开浏览器访问：[http://localhost:8000](http://localhost:8000/)

看到返回的 JSON：

```json
{"message": "Hello, World!"}
```

## 第八步：添加测试

创建测试文件 `test_main.py`：

python

```
# test_main.py
from fastapi.testclient import TestClient
from main import app

client = TestClient(app)

def test_read_root():
    response = client.get("/")
    assert response.status_code == 200
    assert response.json() == {"message": "Hello, World!"}

def test_health_check():
    response = client.get("/health")
    assert response.status_code == 200
    assert response.json() == {"status": "healthy"}
```

运行测试：

```bash
uv run pytest
```

## 第九步：配置开发工具

### 添加代码格式化配置

创建 `pyproject.toml` 中的工具配置：

```text
[tool.black]
line-length = 88
target-version = ['py312']

[tool.pytest.ini_options]
testpaths = ["."]
python_files = "test_*.py"
```

### 格式化代码

```bash
# 格式化代码
uv run black .

# 或者临时运行（不安装）
uv tool run black .
```

## 第十步：版本控制和分享

### 1. 确认 `.gitignore` 内容

bash

```bash
cat .gitignore
```

应该包含：

```text
.venv/
__pycache__/
*.pyc
.DS_Store
```

### 2. 提交到 Git

```bash
git init
git add .
git commit -m "Initial commit with uv project setup"
```
