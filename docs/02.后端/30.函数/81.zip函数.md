## 定义

`zip` 是 Python 的一个内置函数，它的作用就像**拉链**一样，将多个可迭代对象（比如列表、元组、字符串）中**对应位置**的元素，一一配对，打包成一个个元组。

## 基本使用

```python
# 两个列表
names = ["小明", "小红", "小刚"]
scores = [90, 85, 88]

# 使用 zip 打包
result = zip(names, scores)
print(list(result))
# 输出：[('小明', 90), ('小红', 85), ('小刚', 88)]
```

## 返回迭代器

**返回一个迭代器**：`zip` 本身不直接返回列表，而是返回一个迭代器。这可以节省内存。你需要用 `list()` 包裹才能看到全部内容，或者直接用 `for` 循环遍历它。

```python
# 两个列表
names = ["小明", "小红", "小刚"]
scores = [90, 85, 88]

# 使用 zip 打包
result = zip(names, scores)
print(list(result))
# 输出：[('小明', 90), ('小红', 85), ('小刚', 88)]
```

## 最短长度截断

如果传入的序列长度不一致，`zip` 会在最短的序列用完后自动停止

```python
a = [1, 2]
b = ['x', 'y', 'z']
print(list(zip(a, b)))  # 输出：[(1, 'x'), (2, 'y')]  'z' 被忽略了
```

## 支持多个序列

可以同时 `zip` 三个或更多个序列

```python
a = [1, 2, 3]
b = [4, 5, 6]
c = [7, 8, 9]
print(list(zip(a, b, c))) 
# 输出：[(1, 4, 7), (2, 5, 8), (3, 6, 9)]
```

## 并行循环

```python
students = ["小明", "小红", "小刚"]
grades = [90, 85, 88]

for name, grade in zip(students, grades):
    print(f"{name} 的成绩是 {grade}")

# 输出：
# 小明 的成绩是 90
# 小红 的成绩是 85
# 小刚 的成绩是 88
```

## 创建字典

```python
keys = ["name", "age", "city"]
values = ["Alice", 25, "New York"]

my_dict = dict(zip(keys, values))
print(my_dict)  # 输出：{'name': 'Alice', 'age': 25, 'city': 'New York'}
```

## 解压

```python
# 一个由元组组成的列表
paired = [('小明', 90), ('小红', 85), ('小刚', 88)]

# 使用 * 解包，然后 zip 重新组合
names, scores = zip(*paired)

print(names)   # 输出：('小明', '小红', '小刚')
print(scores)  # 输出：(90, 85, 88)
```
