## 列表推导式

### 定义

用一条简洁的语句，从可迭代对象中，生成新列表的语法结构

列表推导式，本质上是对`for 循环  + append() `的一种简写形式

语法格式： `[ 表达式     for    变量    in  可迭代对象  ]`

注意：**没有元组推导式**

### 示例

需求： 让列表中的每个函数，都变为原来的2倍，得到的是一个新的列表

1. 方式一：

   ```python
   nums = [10, 20, 30, 40]
   result = list(map(lambda n: n*2))
   print(result)
   ```

2. 方式二：

   ```python
   nums = [10, 20, 30, 40]
   result = []
   for n in nums:
       result.append(n*2)
   print(result)
   ```

3. 方式三

   ```python
   nums = [10, 20, 30, 40]
   result = [n*2 for n in nums]
   print(result)
   ```

### 带条件的列表推导式

将列表中元素扩大2倍， 大于20的拿出来

```python
nums = [10, 20, 30, 40]
resutl = [n*2 for n in nums if n > 20]
```

## 字典推导式

```python
names = ["张三","李四","王五"]
scores = [60,70,80]
result = {names[index]: scores[index] for index in range(len(names))}
print(result)

#{'张三': 60, '李四': 70, '王五': 80}
```

## 集合推导式

```python
names = ["张三","李四","王五"]
result = {n + '!' for n in names}
print(result)
#{'张三!', '王五!', '李四!'}
```
