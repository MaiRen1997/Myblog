## 导入模块

```python
import random
```

## 基本随机数生成

### 生成 0.0 到 1.0 之间的随机浮点数

```python
# 生成 0.0 到 1.0 之间的随机浮点数（包含0.0，不包含1.0）
num = random.random()
print(num)  # 例如：0.5488135039273248
```

### 生成指定范围内的随机整数

```python
# 生成 a 到 b 之间的随机整数（包含a和b）
num = random.randint(1, 10)  # 生成 1 到 10 之间的整数
print(num)  # 例如：7

# 注意：randint包含两端的值，与range()不同
```

### 2.3 生成指定范围内的随机浮点数

```python
# 生成 a 到 b 之间的随机浮点数
num = random.uniform(1.5, 10.5)  # 生成 1.5 到 10.5 之间的浮点数
print(num)  # 例如：6.3245234523
```

## 序列操作

### 从序列中随机选择一个元素

```python
fruits = ['apple', 'banana', 'orange', 'grape']

# 随机选择一个元素
choice = random.choice(fruits)
print(choice)  # 例如：'banana'

# 随机选择多个元素（不重复）
choices = random.sample(fruits, 2)  # 随机选择2个不同的元素
print(choices)  # 例如：['orange', 'apple']
```

### 打乱序列顺序（原地修改）

```python
cards = ['A', 'K', 'Q', 'J', '10', '9']
random.shuffle(cards)  # 直接修改原列表
print(cards)  # 例如：['J', '10', 'A', '9', 'K', 'Q']
```

## 4. 高级随机数生成

### 正态分布随机数

```python
# 生成符合正态分布的随机数
# mu: 均值，sigma: 标准差
num = random.gauss(mu=0, sigma=1)  # 标准正态分布
print(num)  # 例如：0.345

# 或者使用 normalvariate
num = random.normalvariate(mu=100, sigma=15)  # 均值为100，标准差为15
```

### 其他分布

```python
# 指数分布
num = random.expovariate(lambd=1.0)

# 均匀分布（已介绍的uniform）
# 三角分布
num = random.triangular(low=1, high=10, mode=5)
```

## 随机种子

设置随机种子可以让随机结果可重复（用于调试和测试）：

```python
# 设置随机种子
random.seed(42)  # 可以是任意整数

# 现在每次运行都会得到相同的随机序列
print(random.random())  # 总是：0.6394267984578837
print(random.randint(1, 10))  # 总是：2
```

## 实用示例

### 生成随机密码

```python
import random
import string

def generate_password(length=8):
    characters = string.ascii_letters + string.digits + string.punctuation
    password = ''.join(random.choice(characters) for _ in range(length))
    return password

print(generate_password(10))  # 例如：'aB3$kL9@mN'
```

### 抽奖程序

```python
participants = ['Alice', 'Bob', 'Charlie', 'David', 'Eve']
winners = random.sample(participants, 2)  # 抽取2名不重复的获奖者
print(f"获奖者是：{winners}")
```

### 模拟掷骰子

```python
def roll_dice(times=1):
    results = [random.randint(1, 6) for _ in range(times)]
    return results

print(f"掷一次：{roll_dice()}")
print(f"掷三次：{roll_dice(3)}")
```

## 7. 常用方法总结

| 方法             | 说明                      | 示例                           |
| :--------------- | :------------------------ | :----------------------------- |
| `random()`       | 0.0到1.0的随机浮点数      | `random.random()`              |
| `randint(a, b)`  | a到b的随机整数            | `random.randint(1, 10)`        |
| `uniform(a, b)`  | a到b的随机浮点数          | `random.uniform(1.5, 5.5)`     |
| `choice(seq)`    | 从序列中随机选一个        | `random.choice([1,2,3])`       |
| `sample(seq, k)` | 从序列中随机选k个不重复的 | `random.sample(range(100), 5)` |
| `shuffle(seq)`   | 打乱序列顺序              | `random.shuffle(my_list)`      |
| `seed(x)`        | 设置随机种子              | `random.seed(42)`              |
