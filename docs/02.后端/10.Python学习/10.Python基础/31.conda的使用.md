## 基本使用

1. 创建虚拟环境

   - ```python
     conda create --name myenv python=3.10
     ```

2. 切换/激活幻境

   - ```python
     conda activate 环境名
     ```

3. 退出当前环境

   - ```python
     conda deactivate 环境名
     ```

## 下载依赖包

### 使用pip

1. 安装包

   - ```python
     pip install langchain
     ```

2. 安装指定依赖

   - ```python
     pip install langchain==0.3.7
     ```

3. 批量安装(空格分隔)

   - ```python
     pip install langchain requests numpy
     ```

4. 升级包

   - ```python
     pip install --upgrade langchain
     ```

5. 卸载包

   - ```python
     pip uninstall langchain
     ```

6. 查看已安装包

   - ```python
     pip list
     ```

### 使用conda

1. 安装包

   - ```python
     conda install langchain
     ```

2. 指定频道(如 conda-forge)

   - ```python
     conda install -c conda-force langchanin==0.3.7
     ```

3. 更新包

   - ```python
     conda update langchain
     ```

4. 卸载包

   - ```python
     conda unistall langchain
     ```

5. 查看已安装包

   - ```python
     conda list
     ```
