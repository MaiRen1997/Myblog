## 使用环境变量

1. 在项目根目录创建.env文件

   - ```python
     #.env文件
     OPEN_GPT_URL="https://www.openai.com"
     ```

2. 在py文件中，使用环境变量

   1. 步骤

      1. 引入os 和 dotenv 模块
      2. 用load_dotenv方法，加载配置文件
      3.  通过key 方式，调用环境变量

   2. ```python
      #1. 引入os 和 dotenv 模块
      import os
      import dotenv
      # 2. 调用load_dotenv方法，加载配置文件
      dotenv.load_dotenv()
      # 3. 通过key 方式，调用环境变量
      print(os.getenv('OPEN_GPT_URL'))
      
      ```

   3. 在使用dotenv时，要安装该模块，要使用管理员权限，要进入虚拟环境中才能安装
