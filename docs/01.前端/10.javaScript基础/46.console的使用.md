## log

打印文本信息

### 占位符使用

```js
console.log("圆周率整数部分：%d，带上小数是：%f",3.1415,3.1415);
```

### 设置css样式

```js
console.log('%c这是一个有样式的控制台日志', 'color: white; background-color: blue; padding: 4px;');
```
