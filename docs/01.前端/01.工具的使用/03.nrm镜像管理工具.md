1. 查看镜像

   ```bash
   nrm ls
   ```

2. 切换源

   ```sh
   nrm use taobao #此处暂时以淘宝举例
   ```

3. 添加自定义源

   ```bash
   nrm add myreg https://myregistry.com # 添加自定义源
   ```

4. 删除自定义源

   ```bash
   nrm del myreg
   ```

5. 测试当前源速度

   ```sh
   nrm test npm
   ```
