1. 查看已安装的node版本

   ```bash
   nvm list
   ```

2. 查看可安装的node版本

   ```bash
   nvm ls available
   ```

3. 安装指定版本的node

   ```bash
   nvm install 版本号
   ```

4. 卸载指定版本

   ```sh
   nvm uninstall 版本号
   ```

5. 切换到指定版本

   ```sh
   nvm use 版本号
   ```

6. 显示当前版本

   ```bash
   nvm current
   ```

7. 给不同版本添加别名

   ```sh
   nvm alias
   ```

8. 删除已定义的别名

   ```bash
   nvm unalias
   ```

9. 查看nvm 版本

   ```sh
   nvm version
   ```
