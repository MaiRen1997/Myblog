## 纵横比

### 属性

| Class         | Properties            |
| ------------- | --------------------- |
| aspect-auto   | aspect-ratio: auto;   |
| aspect-square | aspect-ratio: 1 / 1;  |
| aspect-video  | aspect-ratio: 16 / 9; |

### 自定义

如果上述比例不满足，可以支持自定义

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      aspectRatio: {
        '4/3': '4 / 3',
      },
    }
  }
}
```

### 使用插件

```sh
npm install -D @tailwindcss/aspect-ratio
```

```js
// tailwind.config.js
module.exports = {
  theme: {
    // ...
  },
  corePlugins: {
    aspectRatio: false,
  },
  plugins: [
    require('@tailwindcss/aspect-ratio'),
    // ...
  ],
}
```

aspect-w-{n}` and `aspect-h-{n}使用

:::tip

n的值从1~16

:::

## Container

### 用法：

```html
<div class="container mx-auto px-4">
  <!-- ... -->
</div>
```

1. container为容器属性，如果需要加sm,md，可以使用`sm:container`
2. mx-auto，m标识margin，x表示水平方向，将元素的左右外边距设置为自动。这通常用于将块级元素水平居中
3. px-4，p表示padding，x表示水平方向，该属性表示水平内边距

其他属性，也可以使用sm，md等前缀

`md:mx-auto` 表示当屏幕宽度达到或超过 `md` 时，将元素的左右外边距设置为自动。这通常用于将块级元素水平居中，当尺寸小于md时，则mx-auto属性无效

```html
<!-- Full-width fluid until the `md` breakpoint, then lock to container -->
<div class="md:container md:mx-auto">
  <!-- ... -->
</div>
```

### 属性

| 序号 | 属性名称    | 值                 |
| ---- | ----------- | ------------------ |
| 1    | None        | width: 100%;       |
| 2    | sm(640px)   | max-width: 640px;  |
| 3    | md(768px)   | max-width: 768px;  |
| 4    | lg(1024px)  | max-width: 1024px; |
| 5    | xl(1280px)  | max-width: 1280px; |
| 6    | 2xl(1536px) | max-width: 1536px; |

### 自定义配置

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  theme: {
    container: {
      center: true, // 配置元素默认居中
      padding: '2rem', // 配置内边距
      padding: { // 针对不同宽度，配置内边距
        DEFAULT: '1rem',
        sm: '2rem',
        lg: '4rem',
        xl: '5rem',
        '2xl': '6rem',
      },
    },
  },
}
```

## Break After

### CSS属性

break-after 属性规定在指定元素之后是否应发生分页（page-break）、分列（column-break）或分区（region-break）

| 序号 | 属性         | 描述                                                         |
| ---- | ------------ | ------------------------------------------------------------ |
|      | auto         | 默认，在元素后，自动进行分页，分列，分区                     |
|      | all          | 始终在主体框之后立即插入分页符。                             |
|      | always       | 始终在元素之后插入分页符。                                   |
|      | avoid        | 避免在元素之后出现页、列、区域中断                           |
|      | avoid-column | 避免在元素之后分列                                           |
|      | avoid-page   | 避免在元素之后分页                                           |
|      | avoid-region | 避免在元素之后分区                                           |
|      | column       | 始终在元素之后插入分列符                                     |
|      | left         | 在元素之后插入一个或两个分页符，以便将下一页格式化为左页     |
|      | page         | 始终在元素之后插入分页                                       |
|      | recto        | 在主体框后插入一个或两个分页符，以便将下一页格式化为 recto 页 |
|      | region       | 始终在元素之后插入分区符。                                   |
|      | right        | 在元素之后插入一个或两个分页符，以便将下一页格式化为右页     |
|      | verso        | 在主体框后插入一个或两个分页符，以便将下一页格式化为 verso 页 |
|      | initial      | 将此属性设置为其默认值                                       |
|      | inherit      | 从其父元素继承此属性                                         |



### Tailwind CSS属性

| 序号 | 属性                   | 值                     | 说明                                                     |
| ---- | ---------------------- | ---------------------- | -------------------------------------------------------- |
|      | break-after-auto       | break-after:auto       | 默认，在元素后，自动进行分页，分列，分区                 |
|      | break-after-avoid      | break-after:avoid      | 避免在元素之后出现页、列、区域中断                       |
|      | break-after-all        | break-after:all        | 始终在主体框之后立即插入分页符。                         |
|      | break-after-avoid-page | break-after:avoid-page | 避免在元素之后分页                                       |
|      | break-after-page       | break-after:page       | 始终在元素之后插入分页                                   |
|      | break-after-left       | break-after:left       | 在元素之后插入一个或两个分页符，以便将下一页格式化为左页 |
|      | break-after-right      | break-after:right      | 在元素之后插入一个或两个分页符，以便将下一页格式化为右页 |
|      | break-after-column     | break-afer:column      | 始终在元素之后插入分列符                                 |

#### 可以在前面加上hover，focus等

```html
<div class="hover:break-after-column">
  <!-- ... -->
</div>
```

#### 也可以使用媒体查询

```html
<div class="md:break-after-column">
  <!-- ... -->
</div>
```

## Isolation

isolation 属性定义元素是否必须创建新的堆叠内容。

**提示：**与 background-blend-mode 或 mix-blend-mode 一起使用时，isolation 属性很有用。

### CSS属性

| 属性    | 值                                                           |
| ------- | ------------------------------------------------------------ |
| auto    | 默认。只在应用于元素的属性之一需要它时，才创建新的堆叠上下文。 |
| isolate | 必须创建新的堆叠上下文。                                     |

### Tailwind CSS属性

| Class          | Properties          |
| -------------- | ------------------- |
| isolate        | isolation: isolate; |
| isolation-auto | isolation: auto;    |

## Object Fit

object-fit 属性用于规定应如何调整 <img> 或 <video> 的大小来适应其容器。

这个属性告知内容以何种方式填充容器。例如“保留长宽比”或“展开并占用尽可能多的空间”

### CSS属性

| 值         | 描述                                                         |
| :--------- | :----------------------------------------------------------- |
| fill       | 默认值。调整替换内容的大小来填充元素的内容框。如有必要，对象将被拉伸或挤压。 |
| contain    | 缩放被替换的内容以保持其宽高比，同时适合元素的内容框。       |
| cover      | 调整被替换内容的大小，以在填充元素的整个内容框时保持其长宽比。该对象将被裁剪。 |
| none       | 替换的内容不会调整大小。                                     |
| scale-down | 内容的尺寸与 none 或 contain 中的一个相同，取决于它们两个之间谁得到的对象尺寸会更小一些。 |
| initial    | 将此属性设置为其默认值。                                     |
| inherit    | 从其父元素继承此属性。                                       |

### Tailwind CSS属性

| Class             | Properties              |
| ----------------- | ----------------------- |
| object-contain    | object-fit: contain;    |
| object-cover      | object-fit: cover;      |
| object-fill       | object-fit: fill;       |
| object-none       | object-fit: none;       |
| object-scale-down | object-fit: scale-down; |

## Object Position

### CSS属性

默认值50%

| 值         | 描述                                                         |
| :--------- | :----------------------------------------------------------- |
| *position* | 规定图像或视频在其内容框中的位置。第一个值控制 x 轴，第二个值控制 y 轴。可以是字符串（left、center 或 right）或数字（以 px 或 ％ 为单位）。允许负值。 |
| initial    | 将此属性设置为其默认值                                       |
| inherit    | 从其父元素继承此属性。                                       |
