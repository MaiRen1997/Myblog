## 更新系统软件包

```sh
sudo apt update
sudo apt upgrade -y
```

## 安装 Nginx

### 安装nginx

```sh
sudo apt install nginx -y
```

### 启动nginx

- **启动**：`sudo systemctl start nginx`
- **停止**：`sudo systemctl stop nginx`
- **重启**：`sudo systemctl restart nginx`
- **查看状态**：`sudo systemctl status nginx`
- **设置开机自启**：`sudo systemctl enable nginx`
