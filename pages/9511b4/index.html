# 列表list概述

列表不仅可以储存不同的数据类型，而且可以储存大量数据

列表是有序的，有索引值，可切片，方便取值

# 列表创建

# 方式一(常用)

# 方式一：（常用）

# 方式二：（不常用）

# 方式三：列表推导式（后面的课程会讲到）

# 列表的索引切片

l1 = ['a', 'b', '太白', 3, 666]
print(l1[0])  # 'a'
print(l1[-1])  # 666
print(l1[1:3])  # ['b', '太白']
print(l1[:-1])  # ['a', 'b', '太白', 3]
print(l1[::2])  # ['a', '太白', 666]
print(l1[::-1])  # [666, 3, '太白', 'b', 'a']

# sort

(方法用于在原位置对列表进行排序)

# 升序

True表示降序、False表示升序；默认升序reverse=False

list1 = [1, 3, 2, 5]
list1.sort(reverse=False)
print(list1)
1
2
3

# 降序

list1 = [1, 3, 2, 5]
list1.sort(reverse=True)
print(list1)
1
2
3

提示

sort() 的设计思想就是「修改」原列表，而不是返回新的列表；

sorted() 是 sort() 的扩展函数，可以对列表的元素排序，同时不会修改原列表。
