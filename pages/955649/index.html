配置

nuxt.config.ts文件位于Nuxt项目的根目录，可以覆盖或扩展此默认配置

一个最简配置文件导出了defineNuxtConfig函数，函数中包含了一个配置对象。defineNuxtConfig助手函数在全局范围内无需导入即可使用。

export default defineNuxtConfig({
  // 我的Nuxt配置
})
