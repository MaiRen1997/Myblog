# 行程规划

中银： 先预约大行，然后尝试办理，办理不下来，线上申请后，线下找小行申请补签名

9月16出行

  1. 9月14日早上8点线上值机，值机方式， 线上值机后，仍然可以去柜台打印纸质登机牌
    1. 详细版流程 (opens new window)
    2. 重复免费选座 (opens new window)
  2. 4:20点起床去萧山国际机场
  3. 4:50~5:50 萧山国际机场4号航站楼， 25号门
  4. 6:00~6:30 值机(J、K、M窗口)
  5. 8:30~10:50 航程中
  6. 10:50~11:30 到达东荟城
    1. S1巴士乘坐方式 (opens new window)
  7. 11:40~12:00 从东涌巴士总站 (opens new window) -> 東涌東薈城2樓264及273號舖 (opens new window)
