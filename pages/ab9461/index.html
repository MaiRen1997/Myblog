# Object.assign()

  1. Object.assign() 方法用于将所有可枚举属性的值从一个或多个源对象复制到目标对象，它将返回目标对象

  2. 该Api修改第一个参数里的对象

  3. const target = {
    a: 1,
    b: 2
}
const source = {
    b: 4,
    c: 5
}

const returnedTarget = Object.assign(target, source)

console.log(target)
// expected output: Object { a: 1, b: 4, c: 5 }

console.log(returnedTarget)
// expected output: Object { a: 1, b: 4, c: 5 }

# Object.keys

# 返回对象自身的可枚举属性的键组成的数组

# 注意事项

  1. 只返回可枚举属性（enumerable: true）

    const obj = { a: 1, b: 2 };
Object.defineProperty(obj, 'c', { value: 3, enumerable: false });
console.log(Object.keys(obj)); // ['a', 'b']，不包含 'c'
    1
    2
    3

  2. 不返回原型链上的属性

    const parent = { a: 1 };
const child = Object.create(parent);
child.b = 2;
console.log(Object.keys(child)); // ['b']，不包含 'a'
    1
    2
    3
    4

  3. 对 Symbol 属性无效

    const obj = { [Symbol('key')]: 'value' };
console.log(Object.keys(obj)); // []，不返回 Symbol 键
    1
    2

  4. 对数组返回索引字符串

    const arr = ['a', 'b', 'c'];
console.log(Object.keys(arr)); // ['0', '1', '2']

# keys、values、entries注意事项

  1. 传入 null 或 undefined 会报错

    Object.keys(null); // TypeError: Cannot convert undefined or null to object
    1

  2. 对 Map、Set 无效

    const map = new Map([['a', 1], ['b', 2]]);
console.log(Object.keys(map)); // []，应该用 `map.keys()`
    1
    2

  3. 对 String 会拆分成字符

    console.log(Object.keys('abc')); // ['0', '1', '2']
    1

  4. 对 JSON 数据要确保是对象

    const json = '{"a": 1, "b": 2}';
console.log(Object.keys(JSON.parse(json))); // ['a', 'b']
    1

  3. function Person(){

Person.prototype={
  age:3,
  address:"sdag"
}

console.log(Person.prototype.isPrototypeOf(person));//true

# getOwnPropertyNames

  3. var aa = {
    name:"123",

console.log(Object.getOwnPropertyNames(aa))//["name", "age", "idno"]
