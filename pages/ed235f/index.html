# 动画闪烁

使用keyframes关键帧，和animation

@keyframes blink {
  0% {
    opacity: 1; /* 完全不透明 */
  }
  50% {
    opacity: 0; /* 完全透明 */
  }
  100% {
    opacity: 1; /* 再次不透明 */
  }
}
.blinking {
  animation: blink 1s infinite; /* 每1秒闪烁一次，无限次数 */
}
