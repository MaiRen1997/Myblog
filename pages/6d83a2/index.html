# еҜ№иұЎеҰӮдҪ•жүҫеҲ°зұ»зҡ„еұһжҖ§

д№ӢеүҚе’ұд»¬йғҪеӯҰд№ иҝҮпјҢе®һдҫӢеҢ–дёҖдёӘеҜ№иұЎпјҢеҸҜд»ҘйҖҡиҝҮзӮ№зҡ„ж–№ејҸжүҫеҲ°зұ»дёӯзҡ„еұһжҖ§пјҢйӮЈд№Ҳд»–дёәд»Җд№ҲеҸҜд»ҘжүҫеҲ°зұ»дёӯзҡ„еұһжҖ§е‘ўпјҹ

йҖҡиҝҮеӣҫи§ЈиҜҙжҳҺпјҡ

еҜ№иұЎжҹҘжүҫеұһжҖ§зҡ„йЎәеәҸпјҡе…Ҳд»ҺеҜ№иұЎз©әй—ҙжүҫ ------> зұ»з©әй—ҙжүҫ ------> зҲ¶зұ»з©әй—ҙжүҫ ------->.....

зұ»еҗҚжҹҘжүҫеұһжҖ§зҡ„йЎәеәҸпјҡе…Ҳд»Һжң¬зұ»з©әй—ҙжүҫ -------> зҲ¶зұ»з©әй—ҙжүҫ--------> ........

дёҠйқўзҡ„йЎәеәҸйғҪжҳҜеҚ•еҗ‘дёҚеҸҜйҖҶпјҢзұ»еҗҚдёҚеҸҜиғҪжүҫеҲ°еҜ№иұЎзҡ„еұһжҖ§гҖӮ

# зұ»дёҺзұ»д№Ӣй—ҙзҡ„е…ізі»

вјӨеҚғдё–з•Ң, дёҮзү©д№Ӣй—ҙзҡҶжңүи§„еҲҷе’Ңи§„еҫӢ. жҲ‘д»¬зҡ„зұ»е’ҢеҜ№иұЎжҳҜеҜ№вјӨеҚғдё–з•Ңдёӯзҡ„жүҖжңүдәӢзү©иҝӣвҫҸеҪ’зұ». йӮЈдәӢзү©д№Ӣй—ҙеӯҳеңЁзқҖзӣёеҜ№еә”зҡ„е…ізі». зұ»дёҺзұ»д№Ӣй—ҙд№ҹеҗҢж ·еҰӮжӯӨ. еңЁвҫҜеҗ‘еҜ№иұЎзҡ„дё–з•Ңдёӯ. зұ»дёҺзұ»дёӯеӯҳеңЁд»ҘдёӢе…ізі»:

  1. дҫқиө–е…ізі»
  2. е…іиҒ”е…ізі»
  3. з»„еҗҲе…ізі»
  4. иҒҡеҗҲе…ізі»
  5. е®һзҺ°е…ізі»
  6. з»§жүҝе…ізі»(зұ»зҡ„дёүеӨ§зү№жҖ§д№ӢдёҖпјҡз»§жүҝгҖӮ)
