二次封装el-form-item

# el-form-item的二次封装

# 背景：

如果el-form-item里面包含的是下拉框，并且下拉框的下拉选项来自数据字典

需要在某些el-form-item身上显示所用到的数据字典，并能根据配置的关键字，去数据字典配置

# 封装el-form-item

<template>
  <FormItem v-bind="$attrs" v-on="$listeners" :label="label">
    <template slot="label">
      <slot name="label">
        <template v-if="dictCode">
          {{ label }}
          <el-tooltip effect="light">
            <div class="dict-tip" slot="content">
              使用数据字典{{ dictCode }},
              <a href="javascript: void(0)" @click="toDict">去配置</a>
            </div>
            <svg-icon icon-class="dict"></svg-icon>
          </el-tooltip>
        </template>
      </slot>
    </template>
    <slot></slot>
  </FormItem>
</template>

<script>
import { FormItem } from 'element-ui'
export default {
  name: 'ElFormItem',
  props: {
    label: String,
    dictCode: String
  },
  components: {
    FormItem
  },
  methods: {
    toDict() {
      this.$router.push({
        name: 'dictionary',
        params: {
          dictCode: this.dictCode
        }
      })
    }
  }
}
</script>
