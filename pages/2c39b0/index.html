# 使用背景

在Element UI的表单组件中，表单中某一个或者多个字段，根据表单中的一个或者多个字段的值来决定是否必填

# 示例代码

        <el-form-item label="煅烧时间" prop="expiry" :rules="isNecessary('expiry')">
